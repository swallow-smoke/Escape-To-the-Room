# 유니티 2D 타일맵 기반 게임 프로젝트

## 개요
유니티의 2D tilemap을 활용하여 맵 기반 상호작용을 구현하고,
다양한 디자인 패턴과 객체지향 설계 원칙을 실습하기 위해 제작한 프로젝트입니다.

## 주요 기술 스택
- Unity(2D)
- C#
- TilemapSystem
- Custum UI Framework

## 설계 및 아키텍쳐
- Sigleton Pattern
    - UI 매니저, GameManager등 전역 접근이 필요한 매니저 클래스에 적용
- SOLID 원칙 준수
    - 책임 분리 및 의존성 최소화를 고려한 구조를 시도
 
## 핵심 구현 기능
TileMap 기반 인터렉션 시스템
- 2D Tilemap을 기반으로 오브젝트 배치 및 환경 구성
- 인터페이스 기반 설계를 통해 다양한 오브젝트 상호작용 구현
- SOJ를 활용하여 특정 스크립트만 만들면 되게끔 구조 간소화

제네릭 & 상속 구조 활용
- 제네릭을 활용한 타입 안정성 확보 및 코드 재사용성 증가
- 상속 구조를 통해 공통 기능을 추상화 하고 확장 가능한 구조 설계

Attribute 기반 UI 자동 바인딩 시스템
- Reflection + Attribute를 활용한 UI 자동 연결 구조 구현
- UI 요소를 코드에서 직접 연결하지 않고 선언만으로 바인딩 처리
- UIManager에서 Attribute를 탐색하여 자동으로 UI 연결
